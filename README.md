# براتشو كار

نسخة مطورة من مشروع براتشو كار بواجهة أكثر احترافية وهوية أقرب لتطبيقات السوق المفتوح، مع تهيئة Firebase على مشروع `bratsho-car`.

## أهم ما تم تحسينه
- هيدر فخم بهوية براتشو كار
- شريط سفلي احترافي بأيقونات `lucide-react`
- صفحة رئيسية مختصرة وأوضح بصريًا
- إضافة `TopActions` و `ThemeToggle`
- توحيد إعدادات Firebase على مشروع `bratsho-car`

## ملاحظات
- تأكد من تفعيل Google و Phone داخل Firebase Authentication.
- الدردشة الحالية ما تزال واجهة أساسية وتحتاج ربطًا فعليًا مع Firestore لاحقًا.
