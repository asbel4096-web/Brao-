# براتشو كار - مشروع Next.js + Firebase + Tailwind + PWA

هذه نسخة منظمة لمشروع **براتشو كار** بهيكل قريب من منصات الإعلانات الكبيرة، ومجهزة كبداية قوية لتطوير منصة سيارات وقطع غيار وخدمات.

## الموجود داخل المشروع

- Next.js App Router
- Tailwind CSS
- PWA Manifest
- صفحات عامة: الرئيسية، الإعلانات، التفاصيل، إضافة إعلان، المفضلة، الرسائل، الحساب، الاشتراكات
- لوحة إدارة: رئيسية، إدارة الإعلانات، المستخدمين، الاشتراكات
- تكامل Firebase مبدئي
- نقاط API تجريبية داخل Next.js
- ملفات قواعد Firestore وStorage كبداية

## تشغيل المشروع محليًا

```bash
npm install
npm run dev
```

## متغيرات البيئة

انسخ `.env.example` إلى `.env.local` ثم ضع بيانات Firebase:

```bash
cp .env.example .env.local
```

## الخطوات التالية لتحويله إلى منصة حقيقية

1. إنشاء مشروع Firebase
2. تفعيل Authentication
3. تفعيل Firestore
4. تفعيل Storage
5. وضع مفاتيح Firebase في `.env.local`
6. ربط نموذج `add-listing` بالحفظ الفعلي
7. رفع الصور إلى Storage
8. قراءة الإعلانات من Firestore بدل البيانات التجريبية
9. تفعيل صلاحيات المشرف للإدارة
10. إضافة الدفع والاشتراكات والرسائل الفعلية
